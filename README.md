# Starbucks Landing Page Clone
A visually appealing, landing page concept for Starbucks, featuring a clean design, product showcases, and interactive elements.

# Features:
- Drink Switcher: Interactive thumbnails at the bottom that allow users to toggle between different beverage varieties.
- Social Connectivity: Floating social media icons (Facebook, Twitter, Instagram) for quick brand engagement

# Tech Used
- HTML
- CSS
